URL Shortener Microservice

Create a Short URL

URL Shortener

Example Usage:

  • POST a URL to /api/shorturl to get a JSON response with original_url and short_url properties:

    { original_url : 'https://example.com', short_url : 1 }

  • Visit /api/shorturl/1 to be redirected to the original URL.

This project is part of the freeCodeCamp APIs and Microservices certification.
